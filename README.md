# Nefrologia UTI — PWA

## Como publicar no GitHub Pages

1. Acesse github.com e crie uma conta (gratuita)
2. Clique em "New repository"
3. Nome: `nefro-uti` — marque "Public" — clique "Create repository"
4. Clique em "uploading an existing file"
5. Arraste todos os arquivos desta pasta
6. Clique "Commit changes"
7. Vá em Settings → Pages → Source: "Deploy from branch" → Branch: main → Save
8. Aguarde ~2 minutos
9. Acesse: https://SEU-USUARIO.github.io/nefro-uti

## Como instalar no iPhone

1. Abra o link no Safari
2. Toque no ícone de compartilhar (caixa com seta)
3. "Adicionar à Tela de Início"
4. O app aparece como ícone nativo
